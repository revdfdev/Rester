# Quickstart: Dual-Mode Request Editor

## Overview
The Dual-Mode Editor allows you to toggle between a structured "Form" view (Postman-style) and a "Text" view (Standard `.http` syntax).

## Key Features
1. **Real-time Sync**: Change the method in the Form dropdown, and see it update instantly in the Text view.
2. **Request Navigator**: If your file has multiple requests separated by `###`, use the navigator bar at the top of the editor to switch between them.
3. **Scripting Tabs**: Access "Pre-request" and "Tests" scripts directly in the Form UI.
4. **Environment Selection**: Use the global environment picker to resolve `{{variables}}` in your requests.

## How to Switch Modes
Click the "Mode" toggle in the top-right corner of the editor area to switch between **Form** and **Text**.

## Pro Tip
You can rename a request by updating the first comment line above it in Text mode, or by changing the "Name" field in Form mode.
